Project Description

Visual effects can be used in all kinds of projects - be it film, game or for commercials. This project page focuses on the game VFX I have created thus far and consists of the projects "Fractured Calamity", "Untouched Power" and "Reckless". All of these effects were created solely in Unreal Engine 5.0 utilizing the engine's Niagara System, with the rendered output being slightly adjusted in After Effects. Materials for the VFX were also created and edited in Unreal Engine's Material Editor.

The effects presented on this page are mostly leaning towards being in the realistic fantasy theme with colorful visual representation. My final goal with VFX like in these projects is to create my own visual effects for games and film - having my own personal style that can be recognized at first glance.
