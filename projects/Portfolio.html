Project Description

To showcase both my programming and art projects, I created this website as a space to gather everything in one place. This website was developed with the assistance of an AI model called "v0," but I also overhauled most of the code to align it with my aesthetics and coding style. I used only HTML, CSS, and JavaScript, relying on knowledge I gained during my studies, with occasional online lookups for specific functionalities.

This portfolio is hosted using GitHub Pages, and most brand logos were sourced from PNGEgg. Creating this portfolio website has become one of my favorite projects to date, as I had never programmed a website like this before. It allowed me to "try something new" while applying my existing knowledge.
