Maro's Mayhem

Game C# SFML 2D

Project Description

In "Maro's Mayhem" you play as Maro, the notorious Fisherman from a now forgotten village. Fight off waves of opponents in a vertical scrolling map shooter game. Do not get hit from the onslaught of enemies and beat them before they beat you! But always beware of the monsters the vast sea has to offer. This game is leaning towards being on the more difficult side, so do not worry in case you feel like you can not beat the first level. The further you get in the levels, the harder it gets to dodge incoming enemies and handle the ruthless attacks of bosses. But the reward for overcoming all these obstacles is an entry in the highscore list.

This project was created solely through C# code using SFML for basic hardware bindings. All coding was done by myself, as well as the promotional and ingame art. This was the first project that I coded myself and it was a submission for my MultiMediaProject 1 in the second semester of my studies.
